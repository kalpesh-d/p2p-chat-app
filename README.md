# p2p Chat Application

A p2p chat application built with React, Node.js, Socket.IO, and MongoDB.

## Features

- p2p messaging
- Online/Offline user status
- Message history
- User authentication
- Unread message indicators
- User search functionality
- Responsive design

## Tech Stack

### Frontend
- React 19
- Redux Toolkit
- Socket.IO Client
- TailwindCSS
- React Router DOM
- Axios
- date-fns
- react-hot-toast
- Lucide React (icons)

### Backend
- Node.js
- Express
- MongoDB
- Socket.IO
- JWT Authentication
- bcryptjs
- cookie-parser
- cors
- dotenv

## Prerequisites

- Node.js (v16 or higher)
- MongoDB
- npm or yarn
